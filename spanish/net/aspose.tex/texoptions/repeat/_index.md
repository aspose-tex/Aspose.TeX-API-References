## TeXOptions.Repeat property

Obtiene/establece el indicador que indica si es necesario ejecutar el trabajo TeX dos veces en caso de que, por ejemplo, haya referencias en los archivos TeX de entrada. En general, este comportamiento es útil cuando el motor recopila algunos datos a lo largo del proceso de composición tipográfica y los almacena en un archivo auxiliar, todo en la primera ejecución. Y en la segunda ejecución, el motor de alguna manera usa esos datos.

public bool Repeat { get; set; }
