## PdfSaveOptions.JpegQualityLevel property

Категория «Качество» указывает уровень сжатия изображения. Доступные значения от 0 до 100. Чем меньше указанное число, тем выше степень сжатия и, следовательно, ниже качество изображения. Значение 0 означает самое низкое качество изображения, а 100 — самое высокое.
