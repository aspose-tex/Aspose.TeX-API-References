## PdfSaveOptions.OutlineTreeExpansionLevel property

Указывает, до какого уровня следует расширять структуру документа при просмотре файла PDF. 1 - отображаются только элементы структуры первого уровня, 2 - отображаются элементы структуры только первого и второго уровня, и т.д. По умолчанию 1.

public int OutlineTreeExpansionLevel { get; set; }
